# Lawn Tennis Scoring – Vercel Ready

A single-file React app (Vite + Tailwind + Framer Motion + lucide + jsPDF).

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Create a new GitHub repo and push this folder.
2. Go to https://vercel.com/new and import the repo.
3. Framework preset: **Vite** (auto-detected).
4. Build command: `npm run build`
5. Output directory: `dist`
6. Click **Deploy** – you'll get a public URL.
